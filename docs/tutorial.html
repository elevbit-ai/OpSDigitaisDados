Tutorial — OpS Digitais Dados

por Joaquim Pedro de Morais Filho · usacomment.com · zicutake@mail.ru

1. Instalar e executar

cd C:\Users\franc\OpSDigitaisDados
python -m pip install -r requirements.txt
python app\main.py

Ou abra o EXE: dist\OpSDigitaisDados\OpSDigitaisDados.exe

2. Cadastrar usuário

  1. Preencha Nome completo (obrigatório)
  2. Opcional: documento, e-mail, telefone, observações
  3. Clique em Salvar

3. Registrar impressão digital

  1. Selecione o usuário na lista
  2. Escolha o dedo (etiqueta)
  3. Carregar imagem da digital (PNG/JPG do scanner)
  4. Registrar digital no usuário

O sistema extrai um template biométrico (ORB) e grava no banco SQLite interno.

4. Identificar digital

  1. Carregue uma imagem para verificar
  2. Clique em Identificar no banco
  3. Se o score for suficiente, o usuário é encontrado automaticamente

5. Download (exportar tudo)

Botão Exportar tudo gera um ZIP com:

  • ops_digitais_dados.db — banco completo
  • previews/ — imagens das digitais
  • manifest.json — metadados da exportação

6. Upload (importar)

Botão Importar / Upload restaura o ZIP. O banco atual é salvo como .bak antes da substituição.

7. Gerar EXE

powershell -ExecutionPolicy Bypass -File scripts\build_exe.ps1
