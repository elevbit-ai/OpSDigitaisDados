Biometria local · Banco interno · Export / Import

OpS Digitais Dados

Leitor de impressão digital e registrador de usuários com banco de dados interno no PC, download de tudo salvo e upload para restaurar.

por Joaquim Pedro de Morais Filho

Baixar / Usar no PC Tutorial
Os dados (cadastro + templates biométricos) ficam no computador local (SQLite). Exporte o pacote ZIP para backup/download e use Importar para upload/restauração.

O que o sistema faz

  • Cadastro de usuários (nome, documento, e-mail, telefone, notas)
  • Registro de impressão digital a partir de imagem (scanner/exportação)
  • Identificação 1:N no banco interno
  • Exportar tudo (download ZIP com DB + pré-visualizações)
  • Importar / upload do pacote completo
  • Log de auditoria das operações

Recursos

Banco interno

SQLite em pasta data/ no PC.

Digitais

Templates ORB (OpenCV) armazenados por usuário.

Download

Botão Exportar gera ZIP completo.

Upload

Importar restaura o pacote (com backup .bak).

Download e execução

  1. Repositório GitHub: elevbit-ai/OpSDigitaisDados
  2. EXE: dist/OpSDigitaisDados/OpSDigitaisDados.exe
  3. Ou: python app/main.py
  4. Pasta local: C:\Users\franc\OpSDigitaisDados
